Program ini digunakan untuk menghitung volume tabung dengan menggunakan fungsi (method) yang memiliki parameter jari-jari  dan tinggi.Pengguna diminta untuk memasukkan jari-jari dan tinggi tabung.

Nilai tersebut dikirim ke fungsi hitungVolume() sebagai parameter.
Fungsi mengembalikan hasil perhitungan ke main() dan ditampilkan ke layar.
